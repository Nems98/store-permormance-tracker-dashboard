# store-permormance-tracker-dashboard
Power BI Dashboard Project


## Overview

This project is an interactive **Power BI Store performance Dashboard** developed to analyze business performance using data visualization and business intelligence techniques. It enables users to monitor key performance indicators (KPIs), identify sales trends, evaluate profitability, and make informed business decisions through an intuitive dashboard.

---

## Project Objectives

* Analyze overall sales and profit performance.
* Track sales trends over time.
* Identify top-performing products and customers.
* Compare regional performance.
* Build an interactive dashboard for business decision-making.

---

## Dashboard Features

* 📈 Sales Trend Analysis
* 💰 Profit Analysis
* 👥 Customer Performance
* 📦 Category & Sub-Category Analysis
* 🌍 Regional Sales Analysis
* 🎯 KPI Cards (Sales, Profit, Orders, Profit Ratio)
* 🔍 Interactive Filters (Slicers)
* 📊 Dynamic Charts and Visualizations

---

## Tools & Technologies

* **Power BI Desktop**
* **Power Query**
* **DAX (Data Analysis Expressions)**
* **Microsoft Excel**

## Key Insights

* Sales and profit trends are tracked across different time periods.
* Top-performing customers and products are identified.
* Regional performance comparison helps identify growth opportunities.
* Interactive filters allow users to explore the data dynamically.
* KPI cards provide a quick overview of business performance.
